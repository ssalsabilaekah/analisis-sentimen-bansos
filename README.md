# Analisis Sentimen Bantuan Sosial (Bansos)

## Deskripsi

Proyek ini merupakan bagian dari kegiatan **Praktik Kerja Lapangan (PKL)** yang bertujuan untuk menganalisis sentimen masyarakat terhadap program **Bantuan Sosial (Bansos)** di Indonesia menggunakan pendekatan *text mining* dan *machine learning*.

Data opini masyarakat diolah untuk mengetahui kecenderungan sentimen **positif, negatif, dan netral**, sehingga hasil analisis dapat memberikan insight yang berguna sebagai bahan evaluasi kebijakan sosial.

---

## Tujuan

* Mengetahui persepsi dan sentimen masyarakat terhadap program bansos
* Mengklasifikasikan sentimen opini publik menjadi positif, negatif, dan netral
* Menerapkan metode *machine learning* pada data teks dunia nyata

---

## Metodologi

1. **Data Collection**
   Mengumpulkan data teks opini masyarakat terkait bansos

2. **Text Preprocessing**

   * Cleaning data
   * Case folding
   * Tokenizing
   * Stopword removal
   * Stemming

3. **Feature Extraction**

   * TF-IDF (Term Frequency–Inverse Document Frequency)

4. **Sentiment Classification**

   * Machine Learning Model (Naive Bayes)

5. **Evaluation & Analysis**

   * Analisis distribusi sentimen
   * Interpretasi hasil

## Hasil 

Hasil analisis menunjukkan bahwa opini masyarakat terhadap program bansos didominasi oleh sentimen positif, namun masih terdapat sentimen negatif yang berkaitan dengan isu distribusi dan ketepatan sasaran.

Insight ini menunjukkan bahwa analisis sentimen dapat menjadi pelengkap data statistik dalam evaluasi kebijakan sosial.
